# Práctica 1 - Programación Móvil Introducción a Kotlin.

## Estructura

En `src/practica_1` se incluyen los siguientes archivos:

- `calculator.kt`: Calculadora simple con menú (suma, resta, multiplicación y división).
- `employEvaluation.kt`: Evaluación de desempeño y cálculo de nuevo salario.
- `guessNumber.kt`: Juego para adivinar un número aleatorio con intentos limitados.
- `rockPapperScissors.kt`: Juego de Piedra, Papel o Tijeras contra la computadora.

## Requisitos

- JDK 17+ (o versión compatible con tu configuración del proyecto)
- IntelliJ IDEA (recomendado) o compilador Kotlin CLI

## Ejecución

Cada archivo tiene su propia función `main()`, por lo que se ejecuta de forma independiente.

En IntelliJ IDEA:
1. Abre el archivo deseado.
2. Ejecuta la función `main()` de ese archivo.
